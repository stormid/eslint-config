# eslint-config
Standard eslint configuration for all StormId JavaScript, with defaults for for ES2015, React and Jest.

## Installation
```
npm i -D @stormid/eslint-config
```

In your package.json
```
{
  "eslintConfig": {
    "extends": "@stormid/eslint-config"
  }
}
```

Or .eslintrc

```
{
    extends: '@stormid/eslint-config'
}
```
